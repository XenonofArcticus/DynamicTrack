DESCRIPTION

This header file provides access to devices based on VectorNav's VN-200 family of orientation sensors.

Macro Definition Documentation

#define VNASYNC_OFF   0

Asynchronous output is turned off.

#define VNASYNC_VNGPS   20

Asynchronous output type is GPS Measurements.

#define VNASYNC_VNIMU   19

Asynchronous output type is Calibrated Interial Measurements.

#define VNASYNC_VNINS   22

Asynchronous output type is INS Solution.
